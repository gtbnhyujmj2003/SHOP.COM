功能	實作建議

🧠 四、開發建議
可使用 Firebase 或 Supabase 當作後端來快速建立 MVP

初期只需做出「會員登入、商品列表、購物車、回饋點數顯示」這些核心功能

建議前端語言選用 React Native 對 Web 和 Mobile 較靈活，Flutter UI 更漂亮但偏向純 App 開發
